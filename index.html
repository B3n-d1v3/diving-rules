A mobile app to learn the diving rules issues by World Aquatics

The diving rules app has been created to study and learn the diving rules. As a judge or as a diver, you need to learn the rules. As a judge or as a referee you need to regularly train on the rules and more particularly on the penalty summary. This app has been created to help on this focus.

The current version of the app has been update using the lates version of the World Aquatics rule book issues in feb 2023.

What you’ll find?

✔️ the World Aquatics diving rules - updates from feb 2023

✔️ an easy access to the penalty summary

✔️ a quiz to train & learn on all the penalties

Language Support

The content of the Diving Rules has been translated into multiple languages and proposes the rulebooks translation for all those diving federation:

  • 🇬🇧 English
  • 🇫🇷 French
  • 🇪🇸 Spanish
  • 🇲🇽 Mexican
